# 隱性知識提煉師 初階免費版（implicit-knowledge-extractor-lite）

把你已經會、卻散在腦袋裡的經驗，整理成 AI 用得上的知識庫，並幫你挖出一條你自己沒意識到的判斷標準。

給想讓 AI 更懂自己的一般工作者。涵蓋七層提煉的前四層（第零到第三層），交付的是完整的小成果，不是試用片段。

## 這版幫你做兩件事

1. 把你已經會的，整理成 AI 用得上的知識庫。
2. 挖出一條你自己沒注意到、卻一直在用的判斷標準。

## 適合誰

新手建議從這版開始，比較不會有問題。這版可以很自動地幫你提煉分析，你只要貼一件事，就照流程帶完，不用先學框架。

## 還想往下走

更深的直覺、價值觀、判斷根源，屬於完整七層的[進階版 -advanced](https://github.com/Jiang-Yude/implicit-knowledge-extractor-advanced)，同樣免費下載。

## 安裝

```bash
git clone https://github.com/Jiang-Yude/implicit-knowledge-extractor-lite.git \
  ~/.claude/skills/implicit-knowledge-extractor-lite
```

之後在 Claude Code 或支援 Agent Skills 的工具中說「把我的經驗整理給 AI」「幫我做知識庫」即可觸發。
